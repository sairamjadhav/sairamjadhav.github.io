Drowsiness Detection System

A system to enhance road safety by detecting driver fatigue in real-time. Using computer vision, it monitors eye and facial movements to identify drowsiness. It leverages CNNs like ResNet and VGG for feature extraction, paired with OpenCV for video processing. An audio alert warns the driver when fatigue is detected, aiming to prevent accidents.

Key Tech Stack:

Deep Learning, OpenCV, ResNet, VGG, Transfer Learning

Two-Stage Diabetes Prediction

A project to improve early diabetes diagnosis using a two-stage approach. Ensemble learning with CatBoost and LightGBM classifies patients based on clinical data. The second stage refines predictions with lifestyle factors for higher accuracy. It empowers doctors to identify at-risk patients early for timely interventions.

Key Tech Stack:

Machine Learning, CatBoost, LightGBM

Medical Image Denoising

A project to enhance medical imaging like MRI and CT scans by removing noise. I developed a GAN to learn noise patterns and a diffusion model to improve clarity. The result: clearer images for accurate diagnosis. It showcases generative AI’s potential in medical imaging.

Key Tech Stack:

Deep Learning, Diffusion Model, GAN

Sentiment Analysis

A project to classify Twitter sentiments as positive, negative, or neutral. Using NLP, I preprocessed tweets with tokenization and stemming, then applied a transformer model. It provides insights into public opinion on trending topics, useful for businesses and researchers.

Key Tech Stack:

Machine Learning, NLP

Stock Price Forecasting

A project to predict stock prices using historical data. I used LSTM networks to capture temporal patterns, achieving accurate short-term forecasts. Integrated with a Flask web interface, users can input stock symbols to view predictions, bridging AI with finance.

Key Tech Stack:

Time Series Analysis, LSTM
