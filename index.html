Drowsiness Detection System

The Drowsiness Detection System was designed to enhance road safety by identifying driver fatigue in real-time. Using advanced computer vision techniques, the system monitors eye and facial movements to detect signs of drowsiness. It leverages convolutional neural networks (CNNs) like ResNet and VGG for feature extraction, combined with OpenCV for video processing. The system alerts the driver through audio cues when drowsiness is detected, aiming to prevent accidents caused by fatigue.

Two-Stage Diabetes Prediction

The Two-Stage Diabetes Prediction project aimed to improve early diagnosis of diabetes using a novel approach. In the first stage, I used ensemble learning techniques with models like CatBoost and LightGBM to classify patients based on clinical data. The second stage refined predictions by incorporating additional features like lifestyle factors, achieving higher accuracy. This project demonstrated the power of machine learning in healthcare, helping doctors identify at-risk patients earlier and enabling timely interventions.

Medical Image Denoising

The Medical Image Denoising project focused on improving the quality of medical imaging data, such as MRI and CT scans, by removing noise while preserving critical details. I developed two separate models: a Generative Adversarial Network (GAN) to learn noise patterns and a diffusion model to enhance image clarity. The result was clearer images that aided radiologists in accurate diagnosis. This project showcased the potential of generative AI in advancing medical imaging technology.

Sentiment Analysis

The Sentiment Analysis project analyzed Twitter data to classify user sentiments as positive, negative, or neutral. Using natural language processing (NLP) techniques, I preprocessed tweets with tokenization and stemming, then applied a transformer-based model for classification. The system provided insights into public opinion on trending topics, useful for businesses and researchers. This project highlighted my ability to extract meaningful insights from unstructured text data.

Stock Price Forecasting

The Stock Price Forecasting project aimed to predict future stock prices using historical data. I employed Long Short-Term Memory (LSTM) networks to capture temporal patterns in time-series data, achieving accurate predictions over short horizons. The system was integrated with a web interface using Flask, allowing users to input stock symbols and view forecasts. This project bridged my AI expertise with practical applications in finance, demonstrating the power of deep learning in predictive analytics.
